Leetocde_72_Edit_Distance

Author: Haihao Sun Date: June 28, 2020  9:56:25 Category: Leetcode

link to Leetcode_72_edit_distance

#problem description:

Given two words word1 and word2, find the minimum number of operations required to convert word1 to word2.

You have the following 3 operations permitted on a word:

Insert a character
Delete a character
Replace a character
Example 1:

Input: word1 = “horse”, word2 = “ros”
Output: 3
Explanation:
horse -> rorse (replace ‘h’ with ‘r’)
rorse -> rose (remove ‘r’)
rose -> ros (remove ‘e’)

#problem discussion:
This is a classic 2-dimension DP problem, and we need to find transition function, dp state in order to solve it.
