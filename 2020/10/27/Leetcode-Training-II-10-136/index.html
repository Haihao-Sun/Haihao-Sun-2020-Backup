Problem Description 10. Regular Expression Matching | 8/136 | 1/5

Given an input string (s) and a pattern (p), implement regular expression matching with support for '.' and '*' where:

‘.’ Matches any single character.​​​​
‘*’ Matches zero or more of the preceding element.
The matching should cover the entire input string (not partial).

Problem Solution 10. Regular Expression Matching | 8/136 | 1/5

Problem Comment 10. Regular Expression Matching | 8/136 | 1/5

  • While I did spot this is a DP question, I did not come up with correct transfer function based on p[i-1] == ‘*’
  • 0 time p[i-2], 1 or more p[i-2] (but we have to make sure that p[i-2] is valid to use)
  • otherwise, dp[i-1][j-1] and match
  • O(n*m)
